Question-1: What are the different ways to select and element in the DOM?

We use, 1. getElementById, 2. getElementsByClassName, 3. getElementByTagName, 4. querySelector, 5. querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContext?

innerHTML returns HTML tags, while innerText and textContent only works with text.

Question-3: What is event delegation is DOM?

It's a technique in JavaScript where we attach a single event listener to a parent element instead of attaching it to individual child elements.

Question-4: What is event bubbling is DOM?

As an event occurs on an element (click or key-pressing), it first triggers the event handlers on the element itself, and then bubbles up to its parent elements, continuing up to the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

We use document.createElement() to create an element. Use appendChild(), or append() to add an element. We use removeChild() or remove() to remove an element.
